# Тестирование API продуктов

Проект для автоматизированного тестирования API управления продуктами.

## Структура проекта

- `api.py` - модуль с функциями для работы с API
- `test_main.py` - тесты pytest для проверки функциональности API
- `tests.json` - тестовые данные
- `schema.json` - JSON схема для валидации продуктов

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск тестов

### Запуск всех тестов:
```bash
pytest
```
или
```bash
pytest test_main.py
```

### Запуск с подробным выводом:
```bash
pytest -v
```
или
```bash
pytest test_main.py -v
```

### Запуск конкретного теста:
```bash
pytest test_main.py::TestProductAPI::test_add_correct_product -v
```

### Запуск с выводом print-сообщений:
```bash
pytest -v -s
```
или
```bash
pytest test_main.py -v -s
```

## Описание тестов

Тесты проверяют следующие сценарии:

- **Добавление продуктов**: корректное добавление, обработка некорректных данных
- **Редактирование продуктов**: изменение существующих продуктов
- **Удаление продуктов**: удаление существующих и несуществующих продуктов
- **Валидация данных**: проверка обработки некорректных значений

## API Endpoints

Базовый URL: `http://shop2.qatl.ru/`

- `GET /api/products` - получение всех продуктов
- `GET /api/deleteproduct?id={id}` - удаление продукта
- `POST /api/addproduct` - добавление продукта
- `POST /api/editproduct` - редактирование продукта
